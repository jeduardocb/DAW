¿Qué diferencias y semejanzas hay entre Java y JavaScript?
Java:lenguaje de programacion orientada a objetos,es un lenguaje compilado,se depura en dos fases,fuerte tipado,tiene ambito por bloque,tiene clausuras
JS:Esta basado en objetos,lenguaje interpretado,se depura en una fase,debil tipado,tiene ambito por funcion,tiene clausuras

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
getDate(): Devuelve el día del mes. Número entre 1 y 31
getDay():Devuelve el día de la semana. Entre 0 (domingo) y 6 (sábado)
getTime(): Devuelve los milisegundos transcurridos entre el día 1 de enero de 1970 y la fecha correspondiente al objeto al que se le pasa el mensaje
setDate():Actualiza el día del mes
setHours():Actualiza la hora
toTimeString(): Convierte la parte de tiempo de un objeto Date en una cadena

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
concat():Une dos o más arrays
indexOf():Busca un elemento en el array y devuelve su posición. Busca desde el principio y a partir de la primera posición si no se especifica nada
join(): Junta los elementos de un array en una cadena con un separador – opcional.
pop():Borra el último elemento del array y devuelve su contenido
push(): Añade nuevos elementos al array y devuelve su nueva longitud

¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?
exec:Un método RegExp que ejecuta una búsqueda por una coincidencia en una cadena. Devuelve un array de información
match:Un método String que ejecuta una búsqueda por una coincidencia en una cadena. Devuelve un array de información o null si no existe coincidencia alguna.
split:Un método String que utiliza una expresión regular o una cadena fija para cortar una cadena y colocarlo en un array de subcadenas.
replace:Un método String que ejecuta una búsqueda por una coincidencia en una cadena, y reemplaza la subcadena encontrada con una subcadena de reemplazo.
test: Un método RegExp que verifica una coincidencia en una cadena. Devuelve true o false.
