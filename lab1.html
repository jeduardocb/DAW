Investigacion

URL(Uniform Resource Locator)
Es una direccion de internet, cuando el navegador la encuentra muestra este recurso.

MIME(Multipurpose Internet Mail Extensions)
Es la manera standard de mandar contenido en internet a traves de la red, los tipos MIME espeficifican tipos de datos

Métodos y códigos de errores HTTP
Los codigos de estado HTTP te indican si se ha completado correctamente una solicitud HTTP especifica.
Los metodos permiten comunicar al servidor lo que se quiere realizar con un resource bajo una URL

HTML5
Este es un lenguaje markup que se usa para estructurar y presentar el contenido para la web.

JavaScript
ES un lenguaje de programacion que te permite realizar actividad complejas en una pagina web
ejemplo: controlar archivos multimedia,crear imagenes animadas, etc

php(Hypertext Preprocessor)
Es un lenguaje de programacion que se utiliza para crear paginas web dinamica(esto quiere decir que procesa datos),este ser proceda en servidores,

jsp(JavaServer Pages)
Sirve para crear paginas web dinamica, muy parecido a php, pero este usa el lenguaje de programacion Java

Ajax(Asynchronous JavaScript And XML)
Conjunto de tecnicas de desarrollo web que permiten que las aplicaciones web funcionen de manera asincrona procesando cualquier solicitud en segundo plano
(asincrona:el cliente puede realizar varias peticiones al servidor sin necesidad de esperar por la respuesta de la primera)

¿Cuál es la diferencia entre Internet y la World Wide Web?
El internet es una red global conformado por un sistema de computadoras
La Word Wide Web es una recopilación de información a la que se accede a través de internet

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
Get : solicita una peticion de un recurso especifico
Head: pide una respuesta idéntica a la de una petición GET, pero sin el cuerpo de la respuesta.
Post: se utiliza para enviar una identidad a un recurso especifico
Put: reemplaza todas las representaciones actuales del recurso de destino con la carga útil de la petición.
Patch: es utilizado para aplicar modificaciones parciales a un recurso.
Delete : borra un recurso en específico.
